# 设置页

## 功能概述

用户配置 API Key 和管理本地数据的页面。极简设计，只包含必要操作。

---

## 用户故事

- 作为用户，我希望能填写自己的 DeepSeek API Key，使 AI 整理功能可用
- 作为用户，我希望能清除所有本地数据，恢复到初始状态

---

## 当前需求

### API Key 设置

- 输入框类型：`password`（默认隐藏内容）
- 占位提示：`sk-xxxxxxxxxxxxxxxx`
- 失去焦点（`blur`）时自动保存到 `localStorage`（key: `settings`）
- 提示文案：「Key 保存在本地，不会上传到任何服务器。」
- Key 明文存储于 localStorage，风险由用户自担（个人工具，无服务端代理）

### 清除数据

- 按钮文案：「清除所有数据」，危险色（红色）
- 点击后弹出 `confirm` 确认弹窗：「确定要清除所有数据吗？此操作不可恢复。」
- 确认后：删除所有 `diary_*` 条目、`diary_index`、`settings`；API Key 输入框清空

---

## 数据结构

```js
// localStorage key: "settings"

```
